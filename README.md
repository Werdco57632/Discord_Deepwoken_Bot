IMPORTANT:

You must put in a bot token in the "bot_token" variable in deepwoken_bot.py!
(you can make a bot at https://discord.com/developers/applications)
Optionaly, you can also put a discord server_id in if you want it to sync instantly to your server

This bot stores data localy, it will remember anything from any server! Do not put this bot on servers with people you do not trust!

The files in Jsonifier are ment to generate valid json for use with the bot, races files are unrelated and are used for generating races.
The talents.json stores the Jsonifier output talents list
The characters.json stores all the characters you create with the bot
